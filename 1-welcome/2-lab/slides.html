Data in Trees

Structuring and Navigating Files

Concept Acquisition

  1. Data is stored on a computer disk in files.
  2. Files are stored in directories (or folders).
  3. Files and directories can be structured using a tree.
  4. Each file’s location is defined by its path.

Tool Acquisition

  1. The structure of relative and absolute paths.
  2. UNIX commands
    • pwd
    • ls
    • cd
    • cat

Concept Application

  1. Given a file system tree,
    1. write down the absolute path to a file.
    2. given two file system items, write the relative path from one to the other.
  2. Given a set of file paths, reconstruct the file system tree.
  3. Navigate a file system at the command line using the provided UNIX commands.

A named container that stores data – such as text, images, or programs – on a computer disk. The file name usually ends in an extension (.txt, .png) that indicates the format.

How can we specify the location of a file/directory in the tree?

Path

A description of a file or directory’s location in the file system, written as a sequence of folder names that leads to it, separated by /.1

Absolute path: Starts from the root of the file system, indicated by /. Example: /Documents/Schoolwork/HW1.pdf

What absolute paths are found in our tree?

Return to the board and circle a file or two and ask for someone to provide you the absolute path. Write it on the board (be sure to start with /). Repeat by asking the path to a directory. Circle that directory and write the path next to it (it will start with a / and end with the name of that directory).

  1. \ is used on older Windows computers but newer versions recognize the more standard /.

Path

A description of a file or directory’s location in the file system, written as a sequence of folder names that leads to it, separated by /.

Absolute path: Starts from the root of the file system, indicated by /. Example: /Documents/Schoolwork/HW1.pdf

Relative path: Starts from the current directory and shows how to reach a target. .. means “go up one level”. Example if you’re in Documents, Schoolwork/HW1.pdf

What relative paths are found in our tree?

On the board pick at least two relative paths for them to give you, one that goes from a directory to a file in a subdirectory, and another that requires going “up one level” to a file in a differet directory at the same level. You can indiated the “start” and “end” of the relative path using, say a square and a triangle.

UNIX1 Commands

pwd

Returns your present working directory.

ls

Lists the contents of your present working directory.

  1. UNIX is a family of operating systems that includes Linux and MacOS. Most of the commands work on windows too.

cd

Changes your directory to the path provided (after a space). Examples:

  • cd Homework: (relative) only takes you into the Homework directory if you’re in Documents.
  • cd /Document/Homework: (absolute) takes you into the Homework directory regardless of your pwd.

cat

Prints to the screen the contents of the file at the path provided. Example: cat 127.txt.

Return to DataHub and show them examples of each of the commands. At some point (either now or when reviewing answers to the practice problems), you can show them the tree command.
